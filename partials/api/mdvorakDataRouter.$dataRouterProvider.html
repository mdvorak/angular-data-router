$dataRouterProvider
service in module mdvorakDataRouter

Description

Allows simple configuration of all parts of the data router in one place.

Methods

  • apiPrefix(prefix)

    Configures prefix for default view to resource mapping.

    This is just an alias for $dataApiProvider.prefix(prefix) method, see its documentation for details.

    Parameters
    ParamTypeDetails
    prefixString

    Relative URL prefix, relative to base href.

    Returns
    String

    API URL prefix. It's always normalized absolute URL, includes base href.

  • enabled(enabled)

    Enables or disables the router. May be used to disable the router event handling. Cannot be changed after config phase.

    This method is intended mostly for unit tests.

    Enabled by default.

    Parameters
    ParamTypeDetails
    enabledboolean

    Set to false to disable the router.

    Returns
    Object

    Returns provider.

  • error(config)

    Configures view for error page. Error page is displayed when resource or view template cannot be loaded or any of the resolvables fails.

    This is just an alias for $dataRouterRegistryProvider.error(config) method, see its documentation for details.

    Parameters
    ParamTypeDetails
    configObject

    Configuration object, as in when(config).

  • global(config)

    Sets global configuration for all routes. It is then merged into each view configuration, with view taking precedence.

    This is just an alias for $dataRouterLoaderProvider.global(config), see its documentation for details.

    Parameters
    ParamTypeDetails
    configObject

    Configuration object. Currently only "resolve" key is supported.

    Returns
    Object

    Reference to the provider.

  • redirect(path, redirectTo)

    Forces redirect from one location to another.

    Parameters
    ParamTypeDetails
    pathString

    View to force redirect on. Supports wildcards. Parameters are not supported

    redirectToString

    View path which should be redirected to.

    Returns
    Object

    Returns the provider.

  • when(mediaType, config)

    Configures view for given content type.

    This is just an alias for $dataRouterRegistryProvider.when(mediaType,config) method, see its documentation for details.

    Parameters
    ParamTypeDetails
    mediaTypeStringFunction

    Content type to match.

    configObject

    Configuration object.

Example

angular.module('example', ['mdvorakDataRouter'])
    .config(function configRouter($dataRouterProvider) {
        // URL prefixes
        $dataRouterProvider.apiPrefix('api/');

        // Error route
        $dataRouterProvider.error({
            templateUrl: 'error.html',
            dataAs: 'error'
        });

        // Routes
        $dataRouterProvider.when('application/x.example', {
            templateUrl: 'example.html',
            controller: 'ExampleCtrl'
        });
    });
