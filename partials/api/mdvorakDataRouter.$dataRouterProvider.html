$dataRouterProvider
service in module mdvorakDataRouter

Description

Allows simple configuration of all parts of the data router in one place.

Methods

  • apiPrefix(prefix)

    Configures prefix for default view to resource mapping.

    This is just an alias for $dataApiProvider.prefix(prefix) method, see its documentation for details.

    Parameters
    ParamTypeDetails
    prefixString

    Relative URL prefix, relative to base href.

    Returns
    String

    API URL prefix. It's always normalized absolute URL, includes base href.

  • enabled(enabled)

    Enables or disables the router. May be used to disable the router event handling. Cannot be changed after config phase.

    This method is intended mostly for unit tests.

    Enabled by default.

    Parameters
    ParamTypeDetails
    enabledboolean

    Set to false to disable the router.

    Returns
    Object

    Returns provider.

  • global(config)

    Sets global configuration for all routes. It is then merged into each view configuration, with view taking precedence.

    This is just an alias for $dataRouterLoaderProvider.global(config), see its documentation for details.

    Parameters
    ParamTypeDetails
    configObject

    Configuration object. Currently only "resolve" key is supported.

    Returns
    Object

    Reference to the provider.
