emptyHref
service in module mdvorakDataRouter

Description

Defines behavior when link has empty href attribute. It is complementary to apiHref or ngHref directives.

Usage

as attribute
<A empty-href="{Boolean}">
   ...
</A>
as class
<A class="empty-href: {Boolean};">
   ...
</A>

Directive info

  • This directive executes at priority level 0.

Parameters

ParamTypeDetails
emptyHrefBoolean

Must be either hide or disable. Any other value is ignored and warning is logged.

Example

Usage

    <a api-href="{{links.example.href}} empty-href="hide">Hide when no link is given</a>
    <a api-href="{{links.example.href}} empty-href="disable">Disabled when no link is given</a>
    <a api-href="{{links.example.href}} empty-href="disabled">Same as disable</a>
    <a api-href="{{links.example.href}} empty-href="anything">Always visible and active, since attr is invalid</a>
